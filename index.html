How to Install

  1. Install the Xperience xCloud extension from the Chrome Web Store
  2. Go to xbox.com/play

  4. Enjoy enhanced Xbox Cloud Gaming!

Install on Edge Browser (Android)

  1. Install Microsoft Edge from the Play Store

  2. Enable the extension in Safari settings
  3. Go to xbox.com/play
  4. Sign in with your Microsoft account

Install on webOS 22+

  1. Open the web browser on your LG TV

Troubleshooting

Common Issues

Stream Quality Issues

If you're experiencing poor video quality or stuttering:

  1. Check your internet connection speed (minimum 10Mbps recommended)
  2. Try reducing the video quality in settings
  3. Close other bandwidth-intensive applications
  4. Use a wired connection if possible

Controller Not Detected

If your controller isn't being recognized:

  1. Ensure your controller is properly connected to your device
  2. Try reconnecting the controller
  3. Check if the controller works in other applications
  4. Try using a different USB port or Bluetooth connection

Extension Not Working

If the Xperience xCloud extension isn't functioning:

  1. Make sure you're using a supported browser version
  2. Check if the extension is enabled in your browser
  3. Try reinstalling the extension
  4. Clear your browser cache and cookies

High Input Lag

If you're experiencing significant input delay:

  1. Enable the "Reduce Input Lag" option in settings
  2. Check your network latency
  3. Try connecting to a different Xbox server region if available
  4. Disable any unnecessary browser extensions

Platform-Specific Issues

Chrome/Edge

  • Black Screen: Try disabling hardware acceleration in browser settings
  • Extension Conflicts: Disable other extensions that might interfere with Xperience xCloud

Android App

  • App Crashes: Ensure your Android version is compatible (Android 8.0+)
  • Touch Controls Not Working: Reset the touch control layout in settings

Safari (iOS/iPadOS)

  • Audio Issues: Toggle silent mode on and off
  • Controller Disconnects: Update to the latest iOS/iPadOS version

Contact Support

If you're still experiencing issues after trying the troubleshooting steps above, please contact us through our GitHub issues page or join our Discord community for assistance.

Frequently Asked Questions

What is Xperience xCloud?

Xperience xCloud is a browser extension and app that enhances the Xbox Cloud Gaming (xCloud) experience with additional features, improved video quality, and more customization options.

Is Xperience xCloud free?

Yes, Xperience xCloud is completely free and open-source. However, you still need an Xbox Game Pass Ultimate subscription to access Xbox Cloud Gaming.

Will I get banned for using Xperience xCloud?

No, it's safe to use. Xbox Cloud Gaming developers have confirmed they "don't ban people for using a third-party client or extension." However, as with any third-party tool, use it at your own risk.

Which platforms are supported?

Xperience xCloud supports Windows, macOS, Linux, Android, iOS, iPadOS, Android TV, Meta Quest VR headsets, and LG TVs with webOS 22+.

Do I need an Xbox controller?

While an Xbox controller provides the best experience, Xperience xCloud supports various controllers, touch controls on mobile devices, and even mouse and keyboard on desktop platforms.

Can I use Xperience xCloud for Remote Play?

Yes, Xperience xCloud allows you to use Xbox Remote Play through your web browser, which is not available in the standard Xbox Cloud Gaming web app.

How do I update Xperience xCloud?

Browser extensions typically update automatically. For the Android app, you'll need to download the latest version from our GitHub releases page. We announce updates on our GitHub page and Discord community.

Does Xperience xCloud collect my data?

No, Xperience xCloud does not collect any personal data or telemetry. It's completely privacy-focused and doesn't communicate with any servers except Microsoft's Xbox services.

Can I contribute to the project?

Yes! Xperience xCloud is open-source, and we welcome contributions. Check out our GitHub repository for more information on how to contribute.

How can I report bugs or request features?

You can report bugs or request features through our GitHub issues page. Please check existing issues before creating a new one to avoid duplicates.

Guides

Getting Started

Setting Up Xperience xCloud for the First Time

  1. Choose the right client for your device from our installation page
  2. Install the extension or app following the platform-specific instructions
  3. Go to xbox.com/play and sign in with your Microsoft account
  4. Verify that Xperience xCloud is active (look for the icon in the corner)
  5. Configure your preferences in the Xperience xCloud settings menu

Optimizing Your Stream Quality

To get the best possible streaming experience with Xperience xCloud:

  1. Adjust Video Settings: Open the Xperience xCloud overlay and go to Video settings
  2. Set Bitrate: Start with Auto, or manually set based on your connection (10-15 Mbps for 1080p)
  3. Video Filters: Try different clarity and sharpness filters to improve image quality
  4. Monitor Performance: Enable the performance overlay to monitor stream statistics
  5. Network Optimization: Use a wired connection when possible, or stay close to your Wi-Fi router

Advanced Features

Setting Up Mouse and Keyboard Controls

Xperience xCloud allows you to play Xbox games with mouse and keyboard:

  1. Open the Xperience xCloud overlay while in a game
  2. Go to Controls > Mouse and Keyboard
  3. Enable "Mouse and Keyboard Mode"
  4. Customize key bindings for your preferred layout
  5. Adjust mouse sensitivity and acceleration settings
  6. Save your profile for future use

Creating Custom Touch Controls (Mobile)

For mobile users, you can create custom touch control layouts:

  1. Open a game with touch controls enabled
  2. Tap the "Edit Layout" button in the overlay
  3. Drag, resize, and customize button positions
  4. Add additional buttons or macros as needed
  5. Save your custom layout with a name
  6. Create different layouts for different game genres

Using Remote Play via Browser

Xperience xCloud enables Remote Play through your browser:

  1. Ensure your Xbox console is set up for remote play in the console settings
  2. Go to xbox.com/play with Xperience xCloud installed
  3. Look for the "Remote Play" option in the Xperience xCloud menu
  4. Sign in with the same Microsoft account used on your Xbox
  5. Select your console from the list
  6. Wait for the connection to establish

Game-Specific Guides

Optimizing for First-Person Shooters

For the best experience with FPS games like Halo or Call of Duty:

  • Enable "Reduce Input Lag" in the Performance settings
  • Use the "FPS Optimized" video filter preset
  • If using mouse and keyboard, adjust sensitivity and acceleration
  • Consider using a wired controller or connection for minimal latency

Settings for Racing Games

For racing games like Forza Horizon:

  • Prioritize frame rate over resolution in video settings
  • Enable vibration feedback if your controller supports it
  • Use the "Vivid" color profile for more vibrant visuals
  • If using touch controls, try the racing-specific layout template
